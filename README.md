# Exprbin
## Descrição

O programa recebe uma expressão de lógica binária e mostra a sua
tabela-verdade. Os operadores suportados são, em ordem decrescente de
precedência:

- Parenteses: `()`
- Negação: `~`
- E: `∧` ou `^`
- Ou: `∨`, `v` ou `u`
- Se então: `→` ou `>`
- Se e somente se: `↔` ou `=`
